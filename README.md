# Flight-Operations-Analytics & Forecasting

Overview

This project analyzes domestic flight operations at King Khalid International Airport in Riyadh, Saudi Arabia.

Tools Used

* Power BI
* Minitab
* Statistics
* Time Series Analysis

Key Analysis

* Descriptive Statistics
* T-Test
* Regression Analysis
* ARIMA Forecasting
* Trend Analysis
* KPI Dashboards

Key Insights

* Identified seasonal travel patterns.
* Analyzed passenger and flight traffic trends.
* Evaluated airport operational performance.
* Forecasted future passenger demand and flight volumes.
